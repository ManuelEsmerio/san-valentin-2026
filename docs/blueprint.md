# **App Name**: San Valentín Mágico

## Core Features:

- Simbolic Login: Validate a user based on their 'nickname' and anniversary date.
- Arcade Game: Play a 'snake' game with a romantic and clear background and a goal score that once reached opens the next clue.
- Romantic Trivia: Answer a romantic trivia which upon completion reveals the next clue.
- Emotional Revelation: Show an image and final emotinal message

## Style Guidelines:

- Primary color: Soft pink (#FFC0CB) to evoke tenderness and romance.
- Background color: Very light pink (#FFE4E1) for a gentle, loving atmosphere.
- Accent color: Light red (#F08080) to highlight important elements.
- Font: 'Belleza' (sans-serif) for a sweet and elegant touch; if longer text is anticipated, choose 'Alegreya' for body text
- Use heart-shaped icons and other romantic symbols.
- Subtle animations on the heart shaped snake as it moves
